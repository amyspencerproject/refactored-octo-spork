Connect to SAP Datasphere/HANA Cloud

To connect you will need to log into SAP Datasphere with administrator privileges. In the left navigation, click on Space Management. Create a new database and user in the relevant Space. The Database Username, Host Name, and Password are displayed in the Database User Details and will be used in the form below.

Next you will need to whitelist Precog’s IP Addresses. In the left navigation click on System and then the Configuration panel. Select the IP Allowlist tab and add the following IP Addresses 44.205.118.35 (precog.cloud) and 3.73.24.30 (eu.precog.cloud).

Additionally, you will need to upload Amazon’s security certificate. In a new browser window navigate to the Amazon Root CA 1 certificate. Copy all visible text, including the “BEGIN and END CERTIFICATE” lines. Paste this text into a text editor of your choice. Save this text file as a .pem file called precog. You may need to rename the file with the .pem extension after saving the initial file. Upload this file to the Security tab.
